# Ki-Cad LED Circuit — PCB Design

## Description
Simple LED driver circuit designed in KiCad 8.
VCC → 1kΩ Resistor → LED → GND

## Tools Used
- KiCad 8
- 2-layer PCB
- SMD 0805 components
- Ground plane on B.Cu layer

## Files
- .kicad_sch — Schematic
- .kicad_pcb — PCB Layout
- gerbers/ — Manufacturing files
